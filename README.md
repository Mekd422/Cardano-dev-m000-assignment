## Overview

This project is an introductory assignment for **Aiken smart contract development** on Cardano.  
It demonstrates writing and testing Aiken contracts with basic operations, including:

- Boolean operations
- Integer operations
- Comparison operations
- Trace messages in tests

The goal is to practice **unit testing Aiken contracts** and ensure that logic behaves as expected.
